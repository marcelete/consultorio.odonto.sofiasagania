# Consultorio Odontológico - Dra. Sofía Saganía

Sitio web profesional para el consultorio odontológico de la Dra. Sofía Saganía en Buenos Aires.

## 🌐 Características

- **Diseño responsivo** - Se adapta perfectamente a cualquier dispositivo
- **Multiidioma** - Disponible en Español, Inglés y Portugués
- **SEO optimizado** - Meta etiquetas y estructura semántica
- **Rápido y ligero** - HTML puro sin dependencias pesadas
- **Animaciones suaves** - Transiciones y efectos visuales elegantes
- **Contacto y reservas** - Integración preparada para sistemas de reserva

## 📁 Estructura

```
.
├── index.html          # Página principal
├── CNAME              # Configuración de dominio personalizado
└── img/               # Carpeta para imágenes
    └── sofia.png      # Foto de perfil de la Dra. Saganía
```

## 🎨 Paleta de Colores

- **Primario**: Azul profesional (#0E5B8E)
- **Secundario**: Azul claro (#1B7DB8)
- **Fondo**: Crema (#FDFCFB)
- **Texto**: Gris oscuro (#2C2C2C)

## 🚀 Instalación

1. Clona el repositorio
2. Personaliza los datos en `index.html`:
   - Reemplaza la foto `img/sofia.png`
   - Actualiza Google Analytics ID si deseas
   - Configura el dominio en `CNAME`

3. Sube los cambios a GitHub
